# Finly

Прогрессивное веб‑приложение (PWA) для учета личных финансов с поддержкой offline‑режима и установкой как приложения на мобильные и десктопные устройства.

## О проекте

Finly — это учебный проект, разрабатываемый в рамках кейса «Создание PWA‑приложения для управления личными финансами» (МАИ, 01.03.02 — Прикладная математика и информатика). Приложение позволяет пользователям вести учет доходов и расходов, анализировать финансовые привычки и планировать бюджет даже без подключения к интернету.

## Функциональность

### Основные возможности

- **Операции**
  - Добавление доходов и расходов
  - Сумма, тип, категория, дата/время, комментарий
  - Редактирование и удаление операций

- **История операций**
  - Список всех операций
  - Фильтры по периоду (день/неделя/месяц/произвольный диапазон)
  - Фильтры по категориям
  - Поиск по тексту

- **Категории**
  - Базовый набор категорий (еда, транспорт, жильё и т.д.)
  - Пользовательские категории
  - Цвет и иконка для каждой категории

- **Аналитика**
  - Баланс за выбранный период
  - Суммы расходов по категориям
  - Графики и визуализация данных

### Киллер-фичи

- **Бюджеты и лимиты**
  - Месячный/недельный бюджет по категориям
  - Индикатор прогресса и предупреждения о перерасходе

- **Цели и копилки**
  - Создание финансовых целей
  - Прогресс-бар по каждой цели

- **Повторяющиеся операции**
  - Шаблоны для регулярных платежей
  - Автодобавление по расписанию

- **Импорт/экспорт данных**
  - Экспорт в CSV/JSON
  - Импорт из CSV

- **Мультивалютность**
  - Поддержка нескольких валют
  - Пересчет по курсу

- **Темы и персонализация**
  - Светлая/тёмная тема
  - Настройки интерфейса

### PWA и Offline

- **Работа без интернета**
  - Приложение открывается и работает offline
  - Данные сохраняются локально

- **Установка как приложения**
  - Установка на домашний экран
  - Запуск в «приложенческом» режиме
  - Поддержка мобильных и десктопных устройств

## Технологии

### Frontend

- **Языки:** TypeScript, JavaScript, HTML5, CSS3
- **Фреймворк:** React 18+
- **Роутинг:** React Router
- **UI-библиотека:** Material-UI (MUI) / Chakra UI
- **Управление состоянием:** Zustand / Redux Toolkit
- **Графики:** Chart.js / Recharts

### Хранение данных

- **Локальное хранилище:** IndexedDB
- **Библиотека:** Dexie.js

### PWA

- **Сборщик:** Vite
- **PWA плагин:** vite-plugin-pwa
- **Service Worker:** Workbox
- **Manifest:** Web App Manifest

### Инфраструктура

- **Git:** GitHub
- **Деплой:** Vercel / Netlify
- **CI/CD:** GitHub Actions
