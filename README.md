# Finly

Прогрессивное веб‑приложение (PWA) для учета личных финансов с поддержкой offline‑режима и установкой как приложения на мобильные и десктопные устройства.

## О проекте

Finly — это учебный проект, разрабатываемый в рамках кейса «Создание PWA‑приложения для управления личными финансами» (МАИ, 01.03.02 — Прикладная математика и информатика). Приложение позволяет пользователям вести учет доходов и расходов, анализировать финансовые привычки и планировать бюджет даже без подключения к интернету.

## Функциональность

### Основные возможности

- **Операции**
  - Добавление доходов и расходов
  - Сумма, тип, категория, дата/время, комментарий
  - Редактирование и удаление операций

- **История операций**
  - Список всех операций
  - Фильтры по периоду (день/неделя/месяц/произвольный диапазон)
  - Фильтры по категориям
  - Поиск по тексту

- **Категории**
  - Базовый набор категорий (еда, транспорт, жильё и т.д.)
  - Пользовательские категории
  - Цвет и иконка для каждой категории

- **Аналитика**
  - Баланс за выбранный период
  - Суммы расходов по категориям
  - Графики и визуализация данных

### Киллер-фичи

- **Бюджеты и лимиты**
  - Месячный/недельный бюджет по категориям
  - Индикатор прогресса и предупреждения о перерасходе

- **Цели и копилки**
  - Создание финансовых целей
  - Прогресс-бар по каждой цели

- **Повторяющиеся операции**
  - Шаблоны для регулярных платежей
  - Автодобавление по расписанию

- **Импорт/экспорт данных**
  - Экспорт в CSV/JSON
  - Импорт из CSV

- **Мультивалютность**
  - Поддержка нескольких валют
  - Пересчет по курсу

- **Темы и персонализация**
  - Светлая/тёмная тема
  - Настройки интерфейса

### PWA и Offline

- **Работа без интернета**
  - Приложение открывается и работает offline
  - Данные сохраняются локально

- **Установка как приложения**
  - Установка на домашний экран
  - Запуск в «приложенческом» режиме
  - Поддержка мобильных и десктопных устройств

- **Локальное хранилище:** IndexedDB

### PWA

- **Сборщик:** Vite
- **PWA плагин:** vite-plugin-pwa

### Инфраструктура

- **Git:** GitHub
- **Деплой:** Vercel / Netlify
- **CI/CD:** GitHub Actions

## Архитектура

### Клиентское приложение

- **SPA** с роутингом:
  - `/` — список операций и быстрый ввод
  - `/analytics` — аналитика и графики
  - `/budgets` — бюджеты и лимиты
  - `/goals` — финансовые цели
  - `/settings` — настройки и категории

### Модель данных

```typescript
type OperationType = 'income' | 'expense';

interface Operation {
  id: string;
  type: OperationType;
  amount: number;
  categoryId: string;
  datetime: string; // ISO 8601
  comment?: string;
}

interface Category {
  id: string;
  name: string;
  type: OperationType;
  color?: string;
  icon?: string;
}

interface Budget {
  id: string;
  categoryId?: string; // undefined = общий бюджет
  amount: number;
  period: 'week' | 'month';
  startDate: string;
}

interface Goal {
  id: string;
  name: string;
  targetAmount: number;
  currentAmount: number;
  deadline?: string;
  color?: string;
}

interface Settings {
  defaultCurrency: string;
  theme: 'light' | 'dark';
  defaultPeriod: 'day' | 'week' | 'month';
}
```

### PWA-часть

- **`manifest.webmanifest`:**
  - `name`, `short_name`
  - `start_url`, `display`, `theme_color`, `background_color`
  - иконки разных размеров

- **Service Worker:**
  - кэширование статики (HTML, CSS, JS, иконки)
  - offline-first для shell-части
  - корректная работа при отсутствии сети

## Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/amakedoni/finly.git
cd finly

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка production-версии
npm run build

# Локальный предпросмотр собранной версии
npm run preview
```

## Проверка PWA и offline-режима

1. Откройте приложение в Chrome/Edge
2. Установите приложение через кнопку в адресной строке
3. Отключите сеть (offline в DevTools)
4. Проверьте:
   - приложение запускается
   - видны ранее добавленные операции
   - можно добавлять новые операции
5. Прогоните Lighthouse (PWA) и убедитесь, что основные пункты зелёные
